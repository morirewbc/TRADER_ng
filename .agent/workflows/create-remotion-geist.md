---
description: Create Remotion videos using the Geist design system aesthetic
---
1. Scaffold project: run `npm init -y` and install remotion, tailwindcss, @geist-ui/icons, etc.
2. Create core files: remotion.config.ts, tailwind.config.js, and entry point src/index.tsx with `registerRoot()`.
3. Use official brand assets and Geist typography/colors (never use emojis, use proper icons from `@geist-ui/icons`).
4. Apply spring-based animations from `remotion` using `spring` and `interpolate`.
5. Run `npx remotion studio` to preview and `npx remotion render` to render the video.

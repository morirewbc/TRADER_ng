---
description: React composition patterns that scale, including React 19 API changes
---
1. Component Architecture (HIGH Priority): Avoid boolean props to customize behavior; use composition and compound components.
2. State Management (MEDIUM Priority): Decouple implementation from context, define generic context interface, and lift state up when needed.
3. Implementation Patterns (MEDIUM Priority): Create explicit variant components and use children over render props.
4. React 19 APIs: Avoid `forwardRef`; use `use()` instead of `useContext()`.

---
description: Apply premium UI/UX standards to a component or page
---
1. Run a Design Audit checking for standard colors, static feels, and flat designs.
2. Perform a Foundation Check ensuring framer-motion is installed and fonts like Inter or Outfit are set.
3. Component Refactor (The "Glow Up"): Update containers to glassmorphism, use tracking-tight for typography, add scale transitions to cards, and add feedback states.
4. Motion Pass: Wrap page content in a framer-motion motion.div for fade-in entry, and use LayoutGroup.
